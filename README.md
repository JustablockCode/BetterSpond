# BetterSpond

A better Spond experience.

## Setup

1. Run `npm install` to install dependencies.
2. Run `npm run setup` to initialize the project structure.
3. Run `npm start` to start the server.

Window will pop up with modified spond.

## Making plugins

1. Put config into plugins.json.
2. Make js file in plugins folder.
3. Make logo and put in assets folder.
4. Test by launching the app.
